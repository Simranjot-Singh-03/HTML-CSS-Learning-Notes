HTML & Internet Basics

Internet

The Internet is a vast network that connects computers all over the world. It allows users to share information, communicate, and access resources from anywhere. Websites, emails, videos, and online applications all work using the Internet.

The Internet works using a clientâ€“server model where a client (browser) requests data and a server responds with the required information.

HTTP Protocol

The Hypertext Transfer Protocol (HTTP) is an application layer protocol in the Internet protocol suite model. It is used for distributed, collaborative, and hypermedia information systems.

HTTP defines how messages are formatted and transmitted between a web browser and a web server. Whenever you open a website, your browser uses HTTP or HTTPS to communicate with the server.

HTML

HTML, also known as HyperText Markup Language, allows us to write content in a document just as we would in a file created by a word processor.

Unlike a regular text file, HTML also includes structural and layout information about the content. We literally mark up the text of our document with extra information using tags.

HTML is not a programming language. It is a markup language used to structure web content.

What are Elements?

An element is everything from the starting tag to the ending tag. It includes the content inside the tags. 

<p>This is a paragraph</p>

In the above example, the entire line is called an HTML element.

What are Attributes?

Attributes contain extra information about an element that you don't want to appear in the actual content. They are written inside the opening tag. 

<a href="https://example.com">Visit Site</a>

Here, href is an attribute of the anchor tag.

Common HTML Tags

Tag Description
meta Metadata of data
title Used for the title of the page
a Anchor tag used to redirect from one page to another
iframe Used to open a link inside a page
header Used to contain section headings
pre Displays text in fixed-width font and preserves spaces
p Used to write a paragraph
br Used to break a line inside a paragraph
hr Creates a horizontal line
div Used as a container for HTML elements
form Used to create forms
table Used to create tables in HTML

These tags form the basic building blocks of HTML and are commonly used in almost every webpage.
