HTML Tables

The HTML <table> element is used to display data in rows and columns. Tables are useful when presenting structured information such as records, lists, or reports.

A table consists of multiple elements such as rows, headers, and data cells. Proper table structure improves readability and accessibility.

Tag Description
table Defines a table
thead Groups header content
tbody Groups main content
tr Defines a table row
th Defines header cell
td Defines data cell

Using tables correctly helps in organizing data efficiently and presenting it clearly to users.
