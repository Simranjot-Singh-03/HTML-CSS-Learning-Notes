# Web Development Notes Website

This repository contains a **mini Web Development Notes Website** created while learning the fundamentals of web technologies.  
The goal of this project is to document core concepts of **HTML, CSS, and JavaScript** in a structured and easy-to-understand way.

The website is built using **only HTML, CSS, Bootstrap, and basic JavaScript**, and focuses on clarity rather than advanced frameworks.
