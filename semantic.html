Semantic HTML

Semantic HTML refers to the use of HTML tags that clearly describe their meaning both to the browser and to the developer. These tags make the structure of a web page more readable and meaningful.

Instead of using generic tags like <div> everywhere, semantic elements explain what type of content they contain.

Why Semantic HTML is Important

  • Makes code easier to read and understand
  • Improves accessibility for screen readers
  • Helps search engines understand page structure
  • Makes websites more SEO friendly

Common Semantic Elements

Tag Description
header Represents the header of a page or section
nav Contains navigation links
section Defines a section in a document
article Independent content like blog posts
aside Content related indirectly to main content
footer Footer for a page or section

Using semantic HTML makes your web pages more professional and structured.
