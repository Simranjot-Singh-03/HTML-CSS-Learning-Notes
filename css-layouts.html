CSS Layouts

CSS layouts are used to arrange and position elements on a web page. They help in creating clean, responsive, and well-structured designs.

1. CSS Box Model

The CSS box model describes how elements are structured. Every element consists of content, padding, border, and margin. 

.box {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  margin: 15px;
}

  • Content – actual text or element
  • Padding – space inside the border
  • Border – outline around the element
  • Margin – space outside the border

2. Flexbox Layout

Flexbox is a one-dimensional layout system used to align items in rows or columns. It is very useful for navigation bars and responsive components. 

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.item {
  padding: 20px;
  background-color: lightblue;
}

Common Flexbox properties:

  • display: flex – enables flexbox
  • justify-content – horizontal alignment
  • align-items – vertical alignment
  • flex-direction – row or column

3. CSS Grid Layout

CSS Grid is a two-dimensional layout system that allows creating layouts using rows and columns. It is suitable for complex page structures. 

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}

.grid-item {
  background-color: lightcoral;
  padding: 20px;
  text-align: center;
}

Common Grid properties:

  • display: grid – enables grid
  • grid-template-columns – defines columns
  • grid-template-rows – defines rows
  • gap – space between items

When to Use Which Layout

  • Box Model – spacing and sizing elements
  • Flexbox – simple layouts and alignment
  • Grid – complex layouts with rows and columns
